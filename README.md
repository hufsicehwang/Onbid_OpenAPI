# Onbid_OpenAPI
🤖공공데이터를 활용한 온비드 앱개발 프로젝트🤖

# Outline
- Retrofit을 이용해 통신하여 공공데이터를 나타내 줍니다.
- Firebase를 이용해 로그인 관리를 해줍니다.
- Lottie를 이용해 로딩 애니메이션을 구현 했습니다.
- 데이터를 다양하게 분류하여 효율적으로 관리합니다.
